# Heart Disease Prediction

A machine learning classification project developed using Python and patient health data to predict the likelihood of heart disease. The project includes data cleaning, exploratory data analysis, feature preprocessing, model training, evaluation, and performance comparison across multiple classification algorithms.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Features

* Data preprocessing and cleaning
* Exploratory data analysis (EDA)
* Feature engineering and selection
* Model training and evaluation
* Performance comparison across classification algorithms
* Heart disease risk prediction

## Dataset

The project uses a heart disease dataset containing patient health indicators and clinical attributes.

## Results

Multiple machine learning classification models were trained and evaluated to determine their effectiveness in predicting heart disease outcomes.
